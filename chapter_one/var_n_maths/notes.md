# Variables & Math

## Basic data types

~ char (byte)
~ short (short int)
~ long (short int)
~ double (floating point)

arrays, struct, unions of these basic types,
pointers to them,
& functions that return them.

for:

'printf("%d\t%d\n", fahr, celsius)'

% indicates where argument is to be substituted,
%d = int, %3d = 3-digit wide int, %6.1f = 6 digit & one decimal place float,
\t = tab, \n = newline
